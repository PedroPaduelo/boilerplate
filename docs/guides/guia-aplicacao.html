Boilerplate Fullstack

Guia completo de uso da aplicacao

O que e

Um boilerplate fullstack profissional com backend Fastify + frontend React, incluindo autenticacao JWT, dashboard com graficos, dark/light mode e API documentada com Swagger.

Como acessar

Login

  1. Acesse https://boilerplate-app.ddw1sl.easypanel.host
  2. Voce sera redirecionado para a tela de login
  3. Use as credenciais abaixo
  4. Clique em "Entrar"
  5. Voce sera redirecionado para o Dashboard

Usuarios de teste

TipoEmailSenhaRole
Adminadmin@example.comadmin123ADMIN
Usuariouser@example.comuser123USER

Registro

  1. Na tela de login, clique em "Criar conta"
  2. Preencha nome, email e senha (min 6 caracteres)
  3. Clique em "Criar conta"
  4. Voce sera logado automaticamente e redirecionado ao Dashboard

Funcionalidades

Dashboard

  • KPI Cards — Total de Usuarios, Agentes Ativos, Conversas, Tokens Usados
  • Grafico de Linha — Uso ao longo do tempo (7d, 30d, 90d, 1 ano)
  • Grafico Donut — Distribuicao por agente
  • Grafico de Barras — Atividade por horario e top agentes
  • Tabela de Dados — Lista de usuarios com busca, filtro, ordenacao, paginacao e exportacao CSV
  • Toggle Dark/Light — Alterne entre temas no canto superior direito

Autenticacao

  • Login com email/senha (JWT)
  • Registro de novos usuarios
  • Rotas protegidas (redirect automatico para login)
  • Logout via sidebar
  • Sessao persistida no localStorage

Exemplos com curl

# Login
curl -X POST https://boilerplate-api.ddw1sl.easypanel.host/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@example.com","password":"admin123"}'

# Registro
curl -X POST https://boilerplate-api.ddw1sl.easypanel.host/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Novo Usuario","email":"novo@email.com","password":"123456"}'

# Buscar usuario atual (com token)
curl https://boilerplate-api.ddw1sl.easypanel.host/auth/me \
  -H "Authorization: Bearer SEU_TOKEN_AQUI"

Stack Tecnica

Backend

Fastify 5 + TypeScript + Prisma 6 + PostgreSQL + JWT + Zod + BullMQ + Socket.IO + Swagger

Frontend

React 19 + Vite 7 + TypeScript + Tailwind CSS + shadcn/ui + Zustand + TanStack Query + React Hook Form + Recharts + Framer Motion

Dicas

  • O Swagger UI esta disponivel em /docs — use para testar endpoints interativamente
  • O dark mode e o padrao. O tema e persistido no localStorage
  • A tabela do dashboard suporta exportacao CSV — clique no icone de download
  • Redis esta em modo degradado (sem cache/filas), mas a aplicacao funciona normalmente
  • O backend roda em modo producao com build otimizado via tsup
