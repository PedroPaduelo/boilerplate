# 09 — Compartilhamento por link com TTL temporal

> Status: ESQUELETO (aguardando respostas)

## O que o usuário disse
- Mandar para a pessoa um **link** que ela consegue ver **somente por um tempo
  determinado**.
- O **tempo começa a correr a partir do momento em que a URL é aberta** (1ª abertura),
  não a partir da criação do link.

## Decisões em aberto
- [ ] Token de acesso público anônimo? Ou exige login?
- [ ] Duração configurável por link? Padrões?
- [ ] Comportamento após expirar (bloqueia tudo? mostra mensagem?).
- [ ] O link expõe dashboard, gráfico ou relatório? Com cache?
- [ ] Revogação manual do link.
