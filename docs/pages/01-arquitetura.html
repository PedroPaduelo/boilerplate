Riscos de Infraestrutura

🔴 Crítico

  • Redis Single Point

    Se Redis cair, perde cache, sessões e BullMQ. Sistema em modo degradado.

    Mitigação: Redis Cluster + replicação.

  • PostgreSQL Single Point

    Database outage = sistema inoperante.

    Mitigação: HA + replicação + pool.

🟡 Médio

  • Socket.IO Scaling

    Socket.IO não distribuído nativamente. Precisa adapter Redis.

    Mitigação: @socket.io/redis-adapter.

  • BullMQ Workers Memory

    Workers mantêm conexões abertas. Memory leak pode degradar.

    Mitigação: Graceful restart + monitor.

  • Swagger Exposure

    Se SWAGGER_USER/PASS não configurados, docs públicos.

    Mitigação: Forçar auth em produção.

🟢 Baixo

  • Fastify Performance

    Fastify é rápido, performance suficiente.

    Status: OK

  • Helmet Security

    Configurado com CSP, HSTS, frameguard.

    Status: OK
