⚡ System

GET /health 🔓 Public

Verifica se o servidor está online. Retorna status do Redis e DB.

Response 200: 
{ "status": "ok", "redis": true }
cURL: 
curl http://localhost:3333/health

🔐 Auth

POST /auth/register 🔓 Public

Registra novo usuário com role USER por padrão.

Body: 
{ "email": string, "name": string?,
  "password": string (min 6) }
Response 201: 
{ "token": string, "user": {
  id, email, name, role } }
Errors: 
409: email already exists
400: validation error
cURL: 
curl -X POST http://localhost:3333/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"a@a.com","password":"123456"}'
POST /auth/authenticate 🔓 Public

Autentica usuário e retorna JWT. Atualiza lastLoginAt.

Body: 
{ "email": string, "password": string }
Response 200: 
{ "token": string, "user": {
  id, email, name, role } }
Errors: 
401: invalid credentials
401: account inactive
GET /auth/me 🔒 JWT

Retorna dados do usuário autenticado.

Headers: 
Authorization: Bearer <token>
Response 200: 
{ id, email, name, role,
  isActive, lastLoginAt }

👥 Users

GET /users 👑 Admin

Lista todos os usuários com paginação.

Query: 
page, limit, search
Response: 
{ users: [], total, page, limit }
GET /users/:id 🔒 JWT

Retorna dados de um usuário específico.

Params: 
id: string (userId)
Response: 
{ id, email, name, role, isActive }
POST /users 👑 Admin

Admin cria usuário com qualquer role.

Body: 
{ email, name?, password,
  role: 'ADMIN'|'USER' }
PATCH /users/:id 🔒 JWT

Atualiza usuário. Próprio usuário ou admin.

Body (parcial): 
{ name?, email?, password?,
  isActive? (admin only) }
Errors: 
403: cannot edit another user
409: email already exists
DELETE /users/:id 👑 Admin

Remove usuário permanentemente. CASCATA: deleta todos os posts!

⚠️ Cuidado

Deletar um usuário remove TODOS seus posts (onDelete: Cascade no Prisma).

🔍 Search

GET /api/search 🔓 Public

Full-text search com filtros.

Query params: 
q=string, type=post|page|user
status=draft|published, page=1
limit=10, tags=tag1,tag2
GET /api/search/geo 🔓 Public

Busca por localização geográfica (lat/lon/radius).

GET /api/search/autocomplete 🔓 Public

Autocomplete para typeahead.

GET /api/search/analytics 🔓 Public

Estatísticas de busca.

POST /api/search/index 👑 Admin

Indexa ou atualiza documento.

POST /api/search/bulk-index 👑 Admin

Indexação em lote.

DELETE /api/search/document/:id 👑 Admin

Remove documento do índice.
